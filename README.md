# jkvapil.cz

Personal portfolio of Josef Kvapil — frontend architect, Prague.

One page, two languages (`/` EN, `/cs/` CS). A button is built through six
chapters of frontend craft — architecture, TypeScript, design systems,
accessibility, review & AI, delivery — and ends up wired to a real address.

## Stack

Astro · Tailwind 4 · Instrument Serif & JetBrains Mono · zero framework JS

## Development

```sh
pnpm install
pnpm astro dev
pnpm astro build
```

Deploys to Cloudflare Pages via GitHub Actions on push to `main`.
